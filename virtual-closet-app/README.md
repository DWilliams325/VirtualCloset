Prerequisites: 
    - node.js 
    - npm
    - git

Install dependencies: 
    Move into the client directory 
    - cd client
    - npm install 
    
    Move into server directory 
    - cd ../server
    - npm install

Running the Project:
    Front end:
    - cd client 
    - npm run dev 

    Back end:
    - cd server
    - node index.js
